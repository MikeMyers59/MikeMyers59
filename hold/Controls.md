# Controls.md

## Open & Close an Insert
Use JS to change the image after opening/closing the insert to display the proper action icon.

```html
<a href="#" class="openinsert" >⏩</a>
<a href="#" class="closeinsert" >⏪</a>
```

```css
.closeinsert { content: '⏪' }
.openinsert { content: '⏩' }
```
