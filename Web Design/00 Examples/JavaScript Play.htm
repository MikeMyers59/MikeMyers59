A Button Example

This example uses a button so the browser [IE 5.0 and Netscape 4.0 and greater!] can start an action. In the first case, it will cause the users HTML Editor (commonly Notepad) to open and load this HTML document. In the second case, it will attempt to close the current browser. [IE 5.0 will ask the user to confirm if options are setup for maximum security.] The third button displays an alert. The last button shows how to use a Form tag's action attribute. It will reload this page. With a little practice, anything can be done with buttons!

Error Checking Example

This is an example of how JavaScript can be used to error check the data being entered in a form as you leave the field using the onBlur() event. Click in the box and then click outside. An alert will occur if nothing is entered. Enter a value in the box and then click outside and no alert occurs. Try testing the entry for a range of values before leaving and you now have error checking!
