Internet Explorer 11 End of Life

Internet Explorer 11 is set to retire on June 15th, 2022

EBIT is currently developing and testing solutions to transition off Internet Explorer 11. Please see the table below of applications that we are currently working on.

Application Additional Information
ADSU Active Directory Search Utility
ApplCore All "ora.ebnet.gdeb.com" Applications
Arcmis Automated Radiological Management Information System
BILaunchPad Business Objects InfoView / SAP
CAIL EB7G, EB7GQP, EB3G WIP
CPC Common Parts Catalog
CMS Credential Management System
CAES Computer Aided Engineering System
ENAP EB-NNS Addendum Program
Janus / PTR Change Approvals, Configuration Items, Problem Trouble Reports
Passport Quick ATA, Mainframe (Dynamic, mod5, BIW, Test Region)
SubALPS Submarine Automated Label Plate System

If you believe an application requires Internet Explorer to run and don't see the application listed in the above table, please Test the Application in IE11 Mode first, then fill out the Internet Explorer 11 Retirement Survey.

EBIT will be enabling IE11 Mode compatability within Edge for some applications. There will be no difference in functionality for these applications. Other Applications will work in Chrome or Firefox.

EBIT is currently developing and testing solutions for a clean migration off of Internet Explorer 11. EBIT strongly recommends using Edge before the cutover to help ease the transition. Users should also export their bookmarks or favorites from Internet Explorer and Import them into Edge.

EBIT currently allows users to select their preffered default browser. Please see the Wiki article on Default Browser Settings for instructions.

Some applications will work better in one browser versus another. Please be sure to check the Online Applications page to view the reccomended browser for a given application.

Do you know what browser you are currently using? See the below Button to determine what Browser you are currently using.

Exporting Favorites / Bookmarks from Internet Explorer and Importing to a New Browser

Exporting from Internet Explorer

  1. In the Internet Explorer browser, select View favorites, feeds, and history , or select Alt + C to open Favorites.
  2. Under the Add to favorites menu, select Import and export....
  3. Select Export to a file, and then select Next.
  4. On the checklist of options, select Favorites, and then select Next.
  5. Select the folder that you'd like to export your Favorites from, and then select Next.
  6. Type or browse to the location where you'd like your file exported.
  7. Select Export and then Finish.
  8. Your Favorites will now appear in an htm file in the location you've specified.

Importing to Chrome

  1. In the Chrome browser, at the top right, click on the Menu button .
  2. Select Bookmarks, then Import Bookmarks and Settings.
  3. Select the program you'd like to Import, the default should be Microsoft Internet Explorer.
  4. Uncheck Browsing history and Search engines, leave Favorites/Bookmarks checked.
  5. Click Import, then Done.

Importing to Firefox

  1. In the Firefox browser, at the top right, click on the Menu button .
  2. Click Bookmarks, then click Manage Bookmarks bar at the bottom.
  3. Within the Import Bookmarks File window that opens, navigate to the bookmarks HTM/HTML file you are importing and select the file. (This is the file you exported from Internet Explorer)
  4. Click the Open button. The Import Bookmarks File window will close.
  5. Close the Library window.

Importing to Edge

  1. In Microsoft Edge, select Settings and more then select Settings.
  2. Under Profiles, select Import browser data.
  3. Click Choose what to import then change the Import from selection to be Microsoft Internet Explorer.
  4. Uncheck Browing History, Cookies, and Open Tabs, then click Import.

Testing an Application in IE11 Mode

How to Enable IE11 Mode in Edge

  1. In Microsoft Edge, select Settings and more then select Settings.
  2. Click on the Default Browser tab.
  3. Click the drop-down menu to the right of Allow sites to be reloaded in Internet Explorer Mode then change the setting from Default to Allow.
  4. Restart Microsoft Edge.
  5. Navigate to the site you wish to test.
  6. In the top right, select Settings and more, then click on Reload in Internet Explorer mode.
