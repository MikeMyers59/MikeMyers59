HTML Text Modifiers

Content is placed here in paragraphs!

There are numerous Text Modifier tags. They are used to change the appearance of the text. The following HTML Tags are used to modify the look of the enclosed text:

  • <B>...</B> - used to make text appear as Bold.
  • <BIG>...</BIG> - used to make text appear as Big.
  • <I>...</I> - used to make text appear as Italics.
  • <U>...</U> - used to make text appear as Underlined.
  • <S>...</S> - used to make text appear as Strikethrough.
  • <SMALL>...</SMALL> - used to make text appear as Small.
  • <EM>...</EM> - used to make text appear as Emphasis.
  • <STRONG>...</STRONG> - used to make text appear as Strong.

As you can see by the examples in each line above, there are some tags that do similar modifications to the text. Most of these tags are no longer considered normal usage with Style tags and Cascade Style Sheet available. Though most HTML Editors still use the Text Modifier tags, I recommend learning Style tags or Cascade Style Sheets instead.
