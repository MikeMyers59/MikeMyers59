HTML Headers

The Header HTML Tags are used to create Titles and Subtitles as well as separate the sections of the page. Browser display them in varying sizes, colors and styles to differentiate the from the regular text. The Header tags are:

  • <H1>...</H1> - used to enclose the Level 1 Header.
  • <H2>...</H2> - used to enclose the Level 2 Header.
  • <H3>...</H3> - used to enclose the Level 3 Header.
  • <H4>...</H4> - used to enclose the Level 4 Header.
  • <H5>...</H5> - used to enclose the Level 5 Header.
  • <H6>...</H6> - used to enclose the Level 6 Header.

Numerous attributes can be added to the Header Tags to alter their appearance. Normally, Header tags are used in sequence, but some Web pages use them as text modifiers to alter the appearance of the text within the page. I recommend using Text Modifier tags for this purpose.
