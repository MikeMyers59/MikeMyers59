HTML Tables

Tables are a method of easily displaying boring data so that it will be easier to interpret. Data is displayed in rows and columns much like a spreadsheet. Several tags are required to make tables. The following are required for HTML 3.0 - 3.2 Tables:

  • <TABLE>...</TABLE> - used to enclose the entire table.
  • <CAPTION>...</CAPTION> - used to enclose the Caption, also known as Title, of a table.
  • <TR>...</TR> - used to separate the Table Rows and may include either Table Header or Table Data tags.
  • <TH>...</TH> - used to separate the Table Header, creating the individual cell headers.
  • <TD>...</TD> - used to separate the Table Data, creating the individual cells.

An example of the HTML 3.0 - 3.2 Table follows:

HTML 3.0 - 3.2 Table
Header 1 Header 2
Cell 1-1 Cell 1-2
Cell 2-1 Cell 2-2
Cell 3-1 Cell 3-2

The HTML 4.0 Table adds numerous features. The following tags are used for HTML 4.0 Tables:

  • <TABLE>...</TABLE> - used to enclose the entire table.
  • <CAPTION>...</CAPTION> - used to enclose the Caption, also known as Title, of a table.
  • <THEAD>...</THEAD> - used to enclose the Table Head section of a table which contains Table Rows. This section must precede the Table Body section.
  • <TFOOT>...</TFOOTD> - used to enclose the Table Footer section of a table which contains Table Rows. This section must precede the Table Body section.
  • <TBODY>...</TBODY> - used to enclose the Table Body section of a tablewhich contains Table Rows.
  • <TR>...</TR> - used to separate the Table Rows and includes Table Data tags.
  • <TD>...</TD> - used to separate the Table Data, creating the individual cells.

An example of the HTML 4.0 Table follows:

HTML 4.0 Table
Header 1 Header 2
This is a Table Footer
Cell 1-1 Cell 1-2
Cell 2-1 Cell 2-2
Cell 3-1 Cell 3-2

Tables can also be nested within other tables as data. This can lead to elaborate setups and be a cause of confusion to the Source Generator and the browser. The following is an example of a nested table with no borders:

HTML 4.0 Nested Table
Header 1 Header 2 Table 2
This is a Table Footer
Cell 1-1 Cell 1-2
N-Cell 1-1 N-Cell 1-2
N-Cell 2-1 N-Cell 2-2
N-Cell 3-1 N-Cell 3-2
Cell 2-1 Cell 2-2
Cell 3-1 Cell 3-2

For ease of generation, HTML Tables should be limited to nesting of 2 deep to prevent confusion. View source to see the HTML tags required for this page.
