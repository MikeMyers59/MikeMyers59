Using the Icon Set

The Webmasters have obtained an extensive icon set. All the available icons are listed on this page.

Using the icon set is extremely simple. Create a <span> element and set the class to 'fa fa-iconname'. The icons will adapt automatically with the font styles of the element the <span> is placed within. See our Icon Set page for a listing of all available icons.
