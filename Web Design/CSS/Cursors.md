# Cursors

## Common CSS to Modify Cursors
| Action | CSS |  
| --- | --- |  
| Crosshair | `.cursor-crosshair{` <BR> `  cursor:crosshair;` <BR> `}` |  
| Default | `.cursor-pointer{` <BR> `  cursor:default;` <BR> `}` |  
| Pointer | `.cursor-pointer{` <BR> `  cursor:pointer;` <BR> `}` |  
| Text | `.cursor-text{` <BR> `  cursor:text;` <BR> `}` |
