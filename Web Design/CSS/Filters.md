# Filters

## Color Filtering
```css
.greyscale-max,.grayscale-max {
  filter:grayscale(100%);
}
.greyscale,.grayscale {
  filter:grayscale(75%);
}
.greyscale-min,.grayscale-min {
  filter:grayscale(50%);
}

.sepia {
  filter:sepia(80%);
}
.sepia-max {
  filter:sepia(100%);
}
.sepia-min {
  filter:sepia(50%);
}
```
