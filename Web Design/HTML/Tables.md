# Tables

## Table Container
```html
<TABLE>  </TABLE>
```
A Table contains a collection of Rows and may also have a special Table Header row.
class="alt-rows"

## Table Header 
```html
<TR>
  <TH>Header 1</TH>
  <TH>Header 2</TH>
[  <TH>Header 3</TH >]…
</TR>
```

## Table Row
```html
<TR>
  <TD>Cell 1</TD>
  <TD> Cell </TD>
[  <TD>Cell 3 </TD >]…
</TR>
```
Rows are a collection of Table Cells.

## Table Cell
```html
<TD>  </TD>
```
Table cells are components of a Table Row.
