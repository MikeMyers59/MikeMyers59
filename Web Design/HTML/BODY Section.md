# HTML BODY Section

## Use Comments to Block Page
Use you comments to make block sections within the body. This will make readbility better and locating offending HTML easier.
```html
<!--    =====================     HEADER SECTION         =====================     -->
<!--    =====================     NAVIGATION SECTION     =====================     -->
<!--    =====================     CONTENT SECTION        =====================     -->
<!--    =====================     FOOTER SECTION         =====================     -->
```
