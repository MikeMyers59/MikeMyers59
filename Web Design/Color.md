## RGB/RGBa Format

The alpha channel is a deciml value between 0 and 1.0.

Black with different alpha settings
```html
rgba(0,0,0,0.87);
rgba(0,0,0,0.54);
rgba(0,0,0,0.26);
rgba(0,0,0,0.12);
```
